Description

Evolutionary Impressions of Giza, Venus, and Monument Valley using p5.js

The shape and colors used to construct the image vary based on the impression.

Technical

As a starting point, the canvas is filled with shapes of varying positions, dimensions and opacities. The type of shape (ellipse, triangle or rectangle) is determined by the inspiring image, as is the pool of possible shape colors.

The canvas is then mutated using Gaussian distributions centered on each of the non-inspiration-specific shape parameters (position, dimension, and opacity) with deviations that decrease proportionally to the "Mutation Rate".
