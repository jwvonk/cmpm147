Description

A desert landscape with a canalized river running though it, and a multi-chamber dungeon with an underwater entrance.

Overworld:
"." = sand
"_" = sandstone
"=" = grass
"~" = water

Dungeon:
"#" = wall
"." = floor
"~" = water

If you watch closely, you might see the air bubbles of a sea creature rise to the surface of the water.

Technical

  • An autotiling system that generates a bitstring code based on which sides a tile needs to match and a corresponding lookup table.
  • A natural gradient of tiles using Perlin noise forming a canal.
  • A brute-force approach of placing rooms and finding paths between them under a set of constraints.
